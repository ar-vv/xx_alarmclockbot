# ⏰ Telegram Alarm Clock Bot

Минималистичный бот на aiogram 3: каждый пользователь ставит несколько будильников, бот бережно валидирует ввод, ведет базу в SQLite и навязчиво звонит, пока будильник не подтвержден.

## Функции
- Меню с кнопками: «Создать», «Просмотреть», «Удалить», «Стоп».
- Создание будильника пошагово: выбор дня (кнопки или дата числом), часы/минуты (кнопки или текст), опциональный текст напоминания.
- Листинг и удаление любых будильников пользователя.
- Процесс ALARM: сообщение с кнопкой «ок, увидел»; если молчишь минуту — спам раз в секунду еще минуту, в финале пишет, что ты проспал. Останавливается по кнопке, любому сообщению или кнопке «Стоп».

## Запуск
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Экспортируйте токен бота:
   ```bash
   export TELEGRAM_BOT_TOKEN=ваш_токен
   ```
3. Запустите:
   ```bash
   python main.py
   ```

## Примечания по БД
- Хранилище: SQLite-файл `alarms.db` рядом с приложением.
- При старте бот подхватывает будущие будильники из базы и снова их планирует.

## Подсказка по текстам бота
- Короткие и понятные фразы, немного юмора: «Дзынь-дзынь! Подъем-подъем!», «Будильник выключен, рад был помочь».
