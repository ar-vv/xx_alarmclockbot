# ⏰ Telegram Alarm Clock Bot

Простой и надежный телеграм-бот для установки будильников.

## Возможности

- ✅ Создание нескольких будильников на разные дни и время
- ✅ Просмотр всех активных будильников
- ✅ Удаление будильников
- ✅ Напоминания с текстом (опционально)
- ✅ Умный процесс пробуждения с повторными уведомлениями

## Использование

Отправьте `/start` боту для начала работы.
